# Smarthome

A React app that can be used to access all your smart devices in one place.

## Current Hardware Support
- Vesync
- Govee
- Kasa
- iRobot
- Nest

## Future Support
- Sonos

## Installation
1. run `npm install`
2. Rename `env.example` to `.env` and update any/all API Keys and/or passwords
3. run `npm run start`
